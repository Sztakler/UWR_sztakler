## Zadanie 8

Sprawdzić, czy liczba jest dodatnia czy ujemna możemy za pomocą
```
x >> 31
```
Jednak taki zapis zwróci 0, jeśli x >= 0.
```
(-x >> 31)
```
zwróci 1. jeśli x > 0 i 0, jeśli x <=0.

Zauważmy, że wtedy 
```c
-(x >> 31) | (-x >> 31)
``` 
daje 1, jeśli x > 0, -1, jeśli x < 0 i 0, gdy x = 0.
